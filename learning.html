Pusat Pembelajaran Gratis

Tingkatkan keahlian Anda di bidang teknologi dan bahasa bersama kami. Klik untuk memutar video.

Pengembangan Web & Mobile

Kuasai teknologi frontend dan mobile development terkini.

React + Tailwind

Tujuan Pembelajaran:

Mampu membangun antarmuka web yang modern, responsif, dan sangat dapat disesuaikan dengan menggabungkan kekuatan library React dan framework utility-first Tailwind CSS.

  • Setup Proyek & Pengenalan Komponen
  • Dasar-dasar Styling dengan Tailwind
  • State & Props di React
  • Membuat Komponen Reusable
  • React Hooks (useState, useEffect)
  • Desain Responsif dengan Tailwind
  • Manajemen State Global (Context API)
  • Fetching Data & API Integration
  • Optimasi & Deploy Aplikasi

React Native

Tujuan Pembelajaran:

Mampu mengembangkan aplikasi mobile cross-platform untuk iOS dan Android dari satu basis kode menggunakan JavaScript dan React, serta memahami ekosistem native.

  • Pengenalan & Setup Lingkungan Expo
  • Komponen Inti (View, Text, Image)
  • Styling dengan StyleSheet
  • Navigasi (React Navigation)
  • Menangani Input & Form
  • Bekerja dengan List (FlatList)
  • Integrasi API & State Management
  • Menggunakan API Native (Kamera, Lokasi)
  • Membangun & Deploy ke App Stores

JavaScript

Tujuan Pembelajaran:

Menguasai bahasa pemrograman fundamental untuk web, mulai dari sintaks dasar, manipulasi DOM, hingga konsep-konsep modern seperti ES6+ dan pemrograman asinkron.

  • Variabel, Tipe Data, dan Operator
  • Struktur Kontrol (If, For, While)
  • Fungsi dan Scope
  • Manipulasi DOM
  • Array & Object Methods
  • Dasar-dasar ES6 (let, const, arrow functions)
  • Asynchronous JS (Promise, Async/Await)
  • Konsep 'this' dan Prototypes
  • Modules dan Bundling

AI dalam FinTech

Jelajahi persimpangan antara kecerdasan buatan dan teknologi finansial.

Algoritma Trading AI

Tujuan Pembelajaran:

Memahami dan mampu mengimplementasikan dasar-dasar algoritma trading otomatis menggunakan teknik machine learning untuk menganalisis data pasar dan membuat keputusan trading.

  • Pengenalan Algorithmic Trading
  • Dasar Python untuk Finansial
  • Mengambil Data Pasar dengan API
  • Strategi Trading Sederhana (Moving Average)
  • Backtesting Strategi Anda
  • Pengenalan Machine Learning untuk Prediksi
  • Model LSTM untuk Prediksi Time Series
  • Pengenalan Reinforcement Learning
  • Manajemen Risiko dalam Algo Trading

Bahasa Asing

Perluas wawasan global Anda dengan menguasai bahasa internasional.

Bahasa Inggris

Tujuan Pembelajaran:

Meningkatkan kemahiran berbahasa Inggris secara komprehensif, mencakup percakapan, tata bahasa, kosakata, dan pemahaman untuk keperluan akademik maupun profesional.

  • Percakapan Dasar & Perkenalan
  • Tenses Sederhana (Present, Past, Future)
  • Kosakata Umum Sehari-hari
  • Complex Sentences & Conjunctions
  • Idioms & Phrasal Verbs
  • Latihan Mendengar (Listening)
  • Persiapan TOEFL/IELTS
  • Penulisan Akademik & Bisnis
  • Public Speaking & Presentasi
